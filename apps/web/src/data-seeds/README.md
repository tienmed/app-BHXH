# Google Sheets Pilot Seed

These CSV files are starter data for the pilot workbook.

## Suggested Import Flow

1. Create a Google Sheet workbook named `App BHXH Pilot Knowledge Base`.
2. Create tabs using the same names as the CSV filenames without `.csv`.
3. Import each CSV into the matching tab.
4. Deploy the provided Google Apps Script and point it to the workbook.

## Included Tabs

- `catalog_icd.csv`
- `catalog_cls.csv`
- `catalog_medication.csv`
- `protocol_header.csv`
- `protocol_item.csv`
- `rule_claim_risk.csv`
- `rule_cost_composition.csv`
- `mapping_icd_cls.csv`
- `mapping_icd_medication.csv`
- `import_control.csv`

## Pilot Intent

This seed is only for outpatient internal medicine and is deliberately narrow.
